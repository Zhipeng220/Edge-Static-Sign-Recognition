# Demo

Use `run_demo.py` to call the repository prediction entry point with the packaged model artifacts.

Example:

```bash
python demo/run_demo.py --input demo/sample_images --no-tsne
```

`demo/sample_images/` is kept in Git with `.gitkeep`; add local images there for a smoke run.

A short GIF demonstration will be added to the repository README. No GIF file is currently included in this repository snapshot. If a recorded demo is archived later, record the GitHub Release or Zenodo link in `demo_video_zenodo_link.txt`.
