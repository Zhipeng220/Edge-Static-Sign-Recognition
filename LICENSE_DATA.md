# Data License and Redistribution Notice

This repository's source code is covered by the code license in `LICENSE`. Dataset rights are separate from the code license.

## Raw Datasets

Raw third-party datasets are not redistributed in this repository by default. Users must download each dataset from its original source and comply with that dataset's license, citation requirements, and redistribution terms.

Configured dataset folders:

```text
data/raw_datasets/asl_dataset/
data/raw_datasets/indian_sign_language/
data/raw_datasets/nus_hand_posture/
data/raw_datasets/asl_large_dataset/
```

These folders are staging locations only. They are excluded from normal Git tracking.

## Processed Landmarks

Processed landmark files generated by this repository may include derived data from third-party datasets:

```text
X_data.npy
y_labels.npy
class_mapping.npy
metadata.csv
failed_images.csv
per_class_accounting.csv
dataset_accounting_summary.csv
```

Do not redistribute processed landmark arrays unless the original dataset license permits redistribution of derived data. If redistribution is permitted and the files are large, distribute them through Zenodo or another archival release channel rather than the ordinary Git repository.

## Split Files and Summary Tables

Exact split CSV files and aggregate summary tables are intended for reproducibility review and are generally lightweight. Before public release, confirm that these files do not expose restricted raw filenames or paths from datasets whose licenses forbid redistribution.

## User Responsibilities

Before publishing a release, verify:

- original dataset source URLs;
- original dataset license terms;
- required dataset citations;
- whether raw images can be redistributed;
- whether processed landmarks can be redistributed;
- whether sample-level split files can include filenames or only post-filter sample IDs.

When in doubt, redistribute only code, documentation, model weights, and aggregate metrics, and instruct users to regenerate landmarks from locally downloaded raw datasets.
